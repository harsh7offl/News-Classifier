# outputs/

This folder is auto-generated when you run the training and evaluation scripts.

## Files created here

| File | Generated by | Description |
|------|-------------|-------------|
| `model/` | `src/train.py` | Fine-tuned DistilBERT model weights + tokenizer |
| `logs/` | `src/train.py` | Training logs (also streamed to WandB) |
| `confusion_matrix.png` | `src/evaluate.py` | Normalised confusion matrix on test set |
| `eda_class_distribution.png` | `notebooks/eda.ipynb` | Class balance charts |
| `eda_text_length.png` | `notebooks/eda.ipynb` | Word count distribution per class |
| `eda_top_words.png` | `notebooks/eda.ipynb` | Top words per category |

## Note on model size

The `model/` directory (~250MB) is git-ignored.  
To share your trained model publicly, push it to the **HuggingFace Hub**:

```bash
pip install huggingface_hub
huggingface-cli login
huggingface-cli upload <your-username>/news-classifier ./outputs/model
```

Then update the README to point to your Hub model card.
